Section 2: Changes for ACATS 4.2

Version 4.2 of the ACATS updates version 4.1 with additional tests for features defined in [Ada2012], including corrections from the associated Corrigendum ([TC1-2012]).
It includes 178 new tests to check features including null exclusions, explicitly aliased parameters, interface types, record and array aggregates, extended membership tests, quantified expressions, qualified expressions, preconditions and postconditions, nonreturning procedures, build-in-place, "known-to-be-constrained" checks, overriding indicators, pragma Assert, checking pragmas, aspects Predicate_Failure, Default_Value, Default_Component_Value, Pure, Preelaborate, many representation aspects, Ada.containers, and Unchecked_Unions.
In addition, 231 additional tests were corrected or removed to reflect changes in Ada made by [Ada2012], to better locate errors in B-Tests (which helps the accuracy of grading, especially when using the grading tools), as well as in response to test disputes and ARG issue resolutions.
See Annex A, “Version Description” for lists of added, deleted and modified tests, documentation, and support files.
